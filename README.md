# ФОРМАТИРОВАНИЕ ДОКУМЕНТОВ

Flask-проект для формирования двух типов документов:

1. **Удостоверения** — текущий режим из старой версии проекта.
2. **Дипломы** — формирование бланков дипломов, вкладышей и сводной ведомости.

## Что добавлено

- На главной странице две отдельные кнопки:
  - `Создать дипломы`
  - `Создать удостоверения`
- У каждого раздела отдельная загрузка таблицы.
- У каждого раздела отдельное хранение данных:
  - `data/certificates.json`
  - `data/diplomas.json`
- После формирования документов на странице таблицы есть блок:
  - `Хотите убрать нынешнюю таблицу и загрузить новую?`
  - кнопки `Да, очистить таблицу` и `Нет, оставить`.
- Для дипломов создаётся комплект из трёх нормальных документов:
  - `Дипломы.docx` — все дипломы в одном Word-файле, каждый слушатель с новой страницы;
  - `Вкладыши.docx` — все вкладыши в одном Word-файле, каждый слушатель с новой страницы;
  - `Сводная_ведомость.docx` — общая ведомость по всем слушателям.

## Какой файл загружать для дипломов

В режиме **«Создать дипломы»** нужно загружать Excel-файл итоговой ведомости, например:

```text
Итоговая_ведомость_для_дипломов.xlsx
```

Именно из него берутся ФИО, программа, период обучения, объём часов, оценки по модулям и дата проведения аттестационной комиссии.

Файлы `бланк ДИПЛОМ`, `вкладыш А5` и `СВОДНАЯ ВЕДОМОСТЬ` загружать на сайте не надо — они уже лежат в папке `templates_docs/` как шаблоны.

## Установка

```bash
python -m venv .venv
```

Windows PowerShell:

```bash
.\.venv\Scripts\Activate.ps1
```

CMD:

```bash
.venv\Scripts\activate
```

Установка библиотек:

```bash
pip install -r requirements.txt
```

Запуск:

```bash
python run.py
```

Открыть сайт:

```text
http://127.0.0.1:5000
```

## Основные файлы

```text
app/routes.py                         маршруты сайта и две ветки: дипломы / удостоверения
app/services/document_generator.py    генерация удостоверений
app/services/diploma_generator.py     генерация дипломов, вкладышей и ведомости
app/services/diploma_reader.py        чтение Excel-ведомости для дипломов
templates_docs/blank_template.docx    бланк удостоверения
templates_docs/diploma_template.docx  бланк диплома
templates_docs/diploma_insert_template.docx  вкладыш
templates_docs/diploma_statement_template.docx сводная ведомость
```
